# Learn Computer System with Linux

このリポジトリは、コンピューターシステムで学ぶ抽象的な概念を、Linux環境上で観察・確認するための教材です。

CPU、メモリ、ストレージ、OS、プロセス、ファイルシステム、ユーザー権限、標準入出力、ネットワーク、ログ、セキュリティなどを、単なる用語として覚えるのではなく、Linuxコマンドを使って「見える情報」として扱うことを目的とします。

## この教材で重視すること

- 観察する
- 分解する
- 記号化する
- テキスト化する
- コマンドで確認する
- 自分の言葉で説明する

## 位置づけ

この教材は、Linuxコマンドの暗記を目的としたものではありません。

コンピューターシステムの教科書に出てくる概念を、Linux上で観察できる情報に変換し、そこから意味を読み取る練習を行います。

## 関連する学び

この教材は、次の科目・分野への橋渡しとして位置づけます。

- Linux
- コンピューターシステム
- ネットワーク
- 情報セキュリティ
- クラウド
- データベース

ネットワークやセキュリティを詳しく扱う内容は、必要に応じて別リポジトリとして分離します。

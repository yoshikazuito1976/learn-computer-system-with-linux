# 00_environment

## この章の目的

この章では、Linux環境に入ったときに、まず自分が「どこにいて」「誰として操作していて」「どのようなOS上で作業しているのか」を確認します。

コンピューターシステムを学ぶとき、最初に大切なのは、いきなり操作を始めることではなく、現在の環境を観察することです。

Linuxでは、多くの情報がコマンドの実行結果としてテキストで表示されます。
この章では、そのテキスト情報を読み取り、自分の環境を説明できるようになることを目指します。

---

## 学習の流れ

この章では、次の流れで学習します。

1. 現在の場所を確認する
2. 現在のユーザーを確認する
3. ホスト名を確認する
4. OSの種類を確認する
5. カーネル情報を確認する
6. 現在時刻と稼働時間を確認する
7. 確認した情報を自分の言葉で説明する

---

## 1. 現在の場所を確認する

まず、現在どのディレクトリにいるのかを確認します。

```bash
pwd
```

`pwd` は、現在作業しているディレクトリを表示するコマンドです。

**表示例**

```text
/home/siwuser/learn-computer-system-with-linux
```

この結果から、自分が今どの場所で作業しているのかを確認できます。

---

## 2. 現在のユーザーを確認する

次に、現在どのユーザーとして操作しているのかを確認します。

```bash
whoami
```

**表示例**

```text
siwuser
```

Linuxでは、ユーザーによって実行できる操作の範囲が異なります。
そのため、自分がどのユーザーとして作業しているのかを確認することは重要です。

---

## 3. ホスト名を確認する

現在操作しているコンピューターの名前を確認します。

```bash
hostname
```

**表示例**

```text
localhost.localdomain
```

ホスト名は、ネットワーク上でそのコンピューターを識別するための名前です。

---

## 4. OSの種類を確認する

Linuxには、Ubuntu、AlmaLinux、Debian、Rocky Linuxなど、さまざまなディストリビューションがあります。

次のコマンドで、OSの情報を確認します。

```bash
cat /etc/os-release
```

**表示例**

```text
NAME="AlmaLinux"
VERSION="9.5 (Teal Serval)"
ID="almalinux"
VERSION_ID="9.5"
```

ここから、使用しているLinuxディストリビューションやバージョンを読み取ることができます。

**確認するポイント**

- `NAME`
- `VERSION`
- `ID`
- `VERSION_ID`

---

## 5. カーネル情報を確認する

OSの中心部分であるカーネルの情報を確認します。

```bash
uname -a
```

**表示例**

```text
Linux localhost.localdomain 5.14.0-503.el9.x86_64 #1 SMP PREEMPT_DYNAMIC x86_64 GNU/Linux
```

この結果から、次のような情報を読み取ることができます。

- OSの種類
- ホスト名
- カーネルバージョン
- CPUアーキテクチャ

`/etc/os-release` がディストリビューションの情報を示すのに対して、`uname -a` はカーネルに近い情報を示します。

---

## 6. 現在時刻を確認する

現在の時刻を確認します。

```bash
date
```

**表示例**

```text
Sat May 30 15:20:00 JST 2026
```

サーバー管理では、時刻の確認が重要です。
ログを読むときや、処理がいつ実行されたのかを確認するときに必要になります。

---

## 7. 稼働時間を確認する

システムがどのくらい動き続けているかを確認します。

```bash
uptime
```

**表示例**

```text
15:20:10 up 2 hours, 15 minutes, 1 user, load average: 0.00, 0.01, 0.05
```

ここから、次のような情報を確認できます。

- 現在時刻
- システムの稼働時間
- ログインしているユーザー数
- システム負荷の目安

---

## まとめて確認する

ここまでのコマンドをまとめて実行してみます。

```bash
pwd
whoami
hostname
cat /etc/os-release
uname -a
date
uptime
```

一つひとつのコマンド結果を見ながら、次のことを確認します。

- 自分はどのディレクトリにいるか
- 自分はどのユーザーとして操作しているか
- 操作しているコンピューターの名前は何か
- OSの種類とバージョンは何か
- カーネルの情報はどうなっているか
- 現在時刻は正しいか
- システムはどのくらい稼働しているか

---

## 考えてみよう

次の問いに、自分の言葉で答えてください。

1. `pwd` で確認できる情報は何ですか。
2. `whoami` で確認できる情報は何ですか。
3. `/etc/os-release` には、どのような情報が書かれていますか。
4. `uname -a` と `/etc/os-release` の違いは何ですか。
5. サーバー管理において、時刻や稼働時間を確認することはなぜ重要ですか。

---

## この章で大切なこと

Linuxでは、システムの状態や設定情報をコマンドで確認できます。

重要なのは、コマンド名を暗記することだけではありません。

コマンドの結果として表示されたテキストを読み取り、「今、自分はどのような環境で作業しているのか」を説明できるようになることです。

コンピューターシステムの学習は、まず観察から始まります。

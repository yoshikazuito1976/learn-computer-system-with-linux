# 00_environment

## 本章目标

在本章中，你将学习如何检查当前的 Linux 环境。

在操作计算机系统之前，重要的是先观察当前状态：

- 我现在在哪里？
- 我正在使用哪个用户？
- 我运行的是什么操作系统？
- 这台机器的名称是什么？
- 这个系统已经运行了多久？

在 Linux 中，很多系统信息都会以文本形式显示。
本章的目标是读懂这些文本信息，并用自己的话说明你的环境。

---

## 学习流程

在本章中，你将：

1. 检查当前目录
2. 检查当前用户
3. 检查主机名
4. 检查操作系统信息
5. 检查内核信息
6. 检查当前日期和时间
7. 检查系统运行时长
8. 用自己的话说明当前环境

---

## 1. 检查当前目录

首先，检查你当前所在的目录。

```bash
pwd
```

`pwd` 是 “print working directory” 的缩写。
它会显示你当前工作的目录。

**输出示例**

```text
/home/siwuser/learn-computer-system-with-linux
```

通过这个结果，你可以知道自己在文件系统中的当前位置。

---

## 2. 检查当前用户

接下来，检查你当前使用的用户。

```bash
whoami
```

**输出示例**

```text
siwuser
```

在 Linux 中，不同用户可能具有不同权限。
因此，了解你当前使用的是哪个用户很重要。

---

## 3. 检查主机名

检查你正在操作的计算机名称。

```bash
hostname
```

**输出示例**

```text
localhost.localdomain
```

主机名是用于标识一台计算机的名称，尤其是在网络中。

---

## 4. 检查操作系统信息

Linux 有很多发行版，例如 Ubuntu、AlmaLinux、Debian 和 Rocky Linux。

你可以使用以下命令查看操作系统信息：

```bash
cat /etc/os-release
```

**输出示例**

```text
NAME="AlmaLinux"
VERSION="9.5 (Teal Serval)"
ID="almalinux"
VERSION_ID="9.5"
```

你可以从这个文件中读取 Linux 发行版名称和版本。

**需要重点查看的字段**

- `NAME`
- `VERSION`
- `ID`
- `VERSION_ID`

---

## 5. 检查内核信息

内核是操作系统的核心部分。

你可以使用以下命令查看与内核相关的信息：

```bash
uname -a
```

**输出示例**

```text
Linux localhost.localdomain 5.14.0-503.el9.x86_64 #1 SMP PREEMPT_DYNAMIC x86_64 GNU/Linux
```

从这段输出中，你可以看到如下信息：

- 操作系统类型
- 主机名
- 内核版本
- CPU 架构

`/etc/os-release` 显示的是 Linux 发行版信息。
`uname -a` 显示的是更接近内核层面的信息。

---

## 6. 检查当前日期和时间

检查当前日期和时间。

```bash
date
```

**输出示例**

```text
Sat May 30 15:20:00 JST 2026
```

时间信息在系统管理中非常重要。
例如，在阅读日志时，你需要知道某个事件发生的时间。

---

## 7. 检查系统运行时长

检查系统已经运行了多久。

```bash
uptime
```

**输出示例**

```text
15:20:10 up 2 hours, 15 minutes, 1 user, load average: 0.00, 0.01, 0.05
```

从这段输出中，你可以检查：

- 当前时间
- 系统运行时长
- 已登录用户数
- 系统负载平均值

---

## 一次运行所有命令

尝试将所有命令一起运行。

```bash
pwd
whoami
hostname
cat /etc/os-release
uname -a
date
uptime
```

阅读每条结果，并检查以下要点：

- 我当前在哪个目录？
- 我当前使用的是哪个用户？
- 主机名是什么？
- 操作系统名称和版本是什么？
- 内核版本是什么？
- 当前时间是否正确？
- 系统已经运行了多久？

## 本章要点

在 Linux 中，你可以通过命令观察系统状态。

重要的不只是记住命令名称。

你还需要读懂命令输出的文本，并解释它的含义。

学习计算机系统，始于观察。
